## Featured Projects

### 🔬 AI-Powered Lab Report Generator
> FastAPI · SQLite · AI API Integration · PDF Generation · Role-Based Access Control

A full-stack backend system that automates lab report creation using AI. Built with an offline-first architecture, role-based access (admin/doctor/lab staff), and structured PDF output generation.

**What it does:**
- Accepts structured medical/lab input and generates AI-formatted reports
- Role-based access control with authentication
- Offline-first design — works without a live AI connection by falling back gracefully
- Exports polished PDF reports ready for clinical use
